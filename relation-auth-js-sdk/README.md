# Relation auth js sdk

```
├── README.md
├── apps
│  └── relation-auth-demo // demo
├── packages
│  └── auth //js-sdj
└── turbo.json
```

### Build

To build all apps and packages, run the following command:

```
cd relation-auth-js-sdk
pnpm i
pnpm build
```

### Demo

To develop demo, run the following command:

```
cd apps/relation-auth-demo
pnpm dev
```
